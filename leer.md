En este proyecto creamos la simulación de una tienda electrónica con Mobile first (filosofía de diseño y desarrollo web que prioriza la creación de sitios web para dispositivos móviles (smartphones) antes que para ordenadores de escritorio).

Los breakpoints (medidas de ancho (width) de pantalla específicas) se usa a partir de "x" número del ancho de pantalla. Suele considerarse que a partir del ancho de "768px" se empieza a diseñar para tablets y más. A partir de "1024px" se considera diseñar para computadoras o pantallas grandes. 

La sintaxis es "@media(tamaño de pantalla mínimo o máximo: número) {código}"
